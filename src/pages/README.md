# src/pages

小程序（首次下载）页面（主包）

    |- /components       pages内所需组件
    |- /consult          发现板块（tabBar）
    |- /lottery          摇号板块（摇号结果详情）
    |- /me               我的板块（tabBar）
    |- /posts            分享堂板块（分享堂详情）
    |- /project          楼盘板块（tabBar）
    |- /salesMan         置业顾问模块（置业顾问列表、置业顾问详情）
    |- index.wpy         首页
    |- index2.wpy        最新开盘、正在登记、即将摇号、已摇号
    |- lotteryDetail.wpy 楼盘详情
    |- webview.wpy       部分摇号列表H5
    |- webview2.wpy      小程序与h5交互层
